What is included with your
window cleaning service

These window cleaning prices include. All Windows inside and out. Screens hand washed. Frames washed, window sills wiped clean Lot’s of elbow grease, Attention to Detail and a Smile. Please Note Window tracks can be brushed and vacuumed for $3.00 each window.

Regular panes can be up to 6’x6′ Sliding doors included. Small panes can be up to 12in. x 12in. Extra windows & windows that don’t fit this description priced $0.95 to $15.00 per pane. Homes with more or less windows will be prorated.

Note an exact price will be given before starting job.
